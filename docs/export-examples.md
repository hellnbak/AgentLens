# Export examples

AgentLens includes example collector snippets for:

- Splunk HEC: `examples/exporters/splunk-hec/collector-splunk-hec.yaml`
- Elastic/OpenSearch: `examples/exporters/elastic-opensearch/collector-elastic.yaml`
- S3 JSONL buffer/sync: `examples/exporters/s3/`

These are snippets, not default local config. Add them to the relevant collector pipeline for your deployment.
